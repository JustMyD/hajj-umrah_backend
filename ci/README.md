# CI/CD Configuration

Эта директория содержит конфигурационные файлы для CI/CD пайплайнов.

## Структура

- `.github/workflows/` - GitHub Actions workflows
- `.gitlab-ci.yml` - GitLab CI/CD конфигурация
- `docker/` - Docker образы и конфигурации для CI/CD

## Использование

Конфигурационные файлы CI/CD должны быть размещены здесь для централизованного управления и версионирования.
