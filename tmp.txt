Сущности:
- profiles (профиль пользователя)
- bonuses (бонусы)
- rewards (накопления)
- booking (работа с заявками пользователей)
- tour (работа с туром)
- operator (работа с оператором, не уверен что нужно, вроде логики никакой нет, только получить данные оператора чтобы отобразить их на странице)
- review (отзывы)
- donation (благотворительность)
- support (поддержка по сервису)

Действия:
- Изменение персональных данных
- Получить список избранных туров (пока только данные из бд)
- Получить список сравнения (пока только данные из бд)
- Использование бонусов (у пользователей есть бонусы которые они могут потратить на подарки; доставка подарков осуществляется по адресу указанному пользователем в профиле)
- Использование наград (за покупу тура у нас пользователю начисляются денежные накопления, 1единица-1рубль, эти накопления он может либо вывести себе на счет, либо перевести на благотворительную программу нашего сервиса)
- Получить данные о запросах пользователя (запросы которые пользователь оставлял чтобы туроператор связался с ним; коротка информация о связанном туре и статус запроса - договорились или нет)
- Поиск паломнических туров (получить список туров)
- Показать информацию о туре (детали конкретного тура)
- Показать данные операторов (пока только данные из бд)
- Оставить заявку для связи с оператором (пока 2 типа форм связи - коротка для конкретного тура, данные по туру предзаполнены и обощенная форма если захотели связаться не по конкретному туру, а просто с конкретным оператором, данные по пожеланиям нужно будет заполнить вручную)
  - Обобщенная
  - По конкретному туру
- Купить тур? (не реализовано, планы на будующее)
- Оставить отзыв о купленном туре (только для авторизованных пользователей, которые оставляли заявку о желании связаться с оператором, и у которых есть подтверждение что сделка с туроператором была заключена; факт заключения сделки будет подтверждаться туроператором, также у пользователя будет возможность приложить чек об оплате к заявке чтобы получить бонусы)
- Получить список отзывов от пользователей (карусель на главной странице где будет представлено примерно 6 отзываво(пока не решено какие отбирать), и отдельная страница с отзывами на которой будут показаны все отзывы о сервисе)
  - Все
  - Небольшая часть
- Получить историю отправивших донат (сервис занимается также благотворительной программой - часть прибыли мы переводим на эндаунмент фонд чтобы когда набереться достаточная сумма инвестировать ее и с инвестиционной прибыли отправлять людей в хадж, также у пользователей будет возможность осуществить самостоятельный перевод средств на счет в качестве благотворительности, либо перевести бонусные накопления на этот счет; история отправивших донат касается этого)
- Получить статистику отправивших донат (тоже самое только статистика по пользователям осуществившим пожертвование)
- Получить текущее состояние благотворительности (ну тут состояние эндаунмент фонда, сколько средств накоплено, сколько нужно всего, и инфа о том сколько людей было отправлено в паломничество - умру или хадж)
- Задонатить в эндаунмент фонд (это будет реализовано в будущем, форма для доната)
- Задать вопрос по сервису (форма которую пользователь должен будет заполнить, каждый вопрос должен быть обработан и помечен одним из статусов)
